# Algoritmos de Ordenamiento en PHP

Este repositorio contiene la implementación de tres algoritmos de ordenamiento:
- Bubble Sort: Ordena una lista de números de forma descendente.
- Merge Sort: Ordena una lista de palabras alfabéticamente.
- Insertion Sort: Ordena una lista de nombres alfabéticamente.

## Cómo ejecutar
- Clona el repositorio: `git clone https://github.com/dev-rafael-arevalo/ProblemasOrdenamiento`
- Ejecuta los scripts con: `php <nombre-del-script>.php`
